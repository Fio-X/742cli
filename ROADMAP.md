# 742cli Roadmap (6 Months)

This roadmap defines delivery targets for turning `742cli` into a stable, community-friendly AI workflow project.

## 2026-04: Security and baseline hardening

- finalize security policy and disclosure workflow
- add contributor checklist for secret handling and redaction
- validate CI coverage for core commands and deterministic examples

## 2026-05: Sandbox alpha

- publish initial demo sandbox setup
- document reproducible issue template tied to sandbox state
- collect first round community feedback and fix high-priority friction

## 2026-06: Workflow expansion

- add templates for issue triage and release preparation
- expand example artifacts for prompt iteration and evaluation
- improve CLI output consistency for screenshot and docs reuse

## 2026-07: Reliability milestone

- strengthen regression checks around workflow rendering
- add guardrails for invalid input and missing context cases
- reduce setup friction for Windows and macOS contributors

## 2026-08: Community onboarding

- publish short tutorials for first-time contributors
- provide a concise maintainer playbook for review and release cadence
- track contributor experience metrics from issues and PRs

## 2026-09: Public milestone release

- tag a documented milestone release with changelog summary
- publish lessons learned from sandbox and security iteration
- define next roadmap cycle based on community adoption data
